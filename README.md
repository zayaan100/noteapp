Notes App – Expo & React Native
The app allows users to create, view, and delete notes using a clean and simple interface

1.  Project Description
The Notes App is designed as a minimal productivity tool. All notes are stored locally on the device, ensuring fast performance and offline usage.

The application contains four main screens:

Add Note – create a new note

Notes – view and delete notes

About – display author details

Settings – theme toggle (Light/Dark)

2. Installation & Run Instructions
Prerequisites
    Node.js (LTS)
    Yarn or npm
    Expo CLI
    Expo Go (Android/iOS)

Install Dependencies
yarn install or npm install
expo start

Features
Add notes
    Users can write a note and save it
View Notes
    A list of all saved notes appears in the Notes tab
Delete notes
    Each note includes a delete option for quick removal.
Theme Toggle
    Switch between Light and Dark mode from Settings.
